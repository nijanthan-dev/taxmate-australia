# Security Policy

## Supported Versions

Only the latest `main` branch is supported.

## Reporting

Do not open a public issue for secrets, credentials, private tax records, or vulnerabilities.

Report security concerns privately to the repository owner through GitHub.

## Data Handling

TaxMate Australia is a preparation aid. Do not commit real tax records, identity documents, TFNs, Medicare numbers, bank details, or private client files.

The repository contains ATO source registry and coverage metadata only. Fetched ATO page text is an ignored local cache under `.cache/ato/`. User records should stay outside the repository.

TaxMate Australia is not affiliated with, sponsored by, endorsed by, or approved by the Australian Taxation Office or any government agency.
